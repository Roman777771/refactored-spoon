# Avito-TON Clone Project

## Overview
This project is a clone of the popular Avito platform, built using the TON blockchain technology. The goal is to provide a decentralized marketplace where users can buy and sell goods and services in a trustless environment.

## Features
- **Decentralization**: Leveraging TON's blockchain for a decentralized experience.
- **User-Friendly Interface**: Intuitive UI for easy navigation and usage.
- **Secure Transactions**: Utilizing smart contracts to ensure secure transactions between buyers and sellers.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Roman777771/refactored-spoon.git
   cd refactored-spoon
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
